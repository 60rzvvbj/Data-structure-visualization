```md
## 括号匹配问题
给定一个只包括`()[]{}`的字符串s，判断字符串是否合法

**有效字符串需满足：**
1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例**
	```
	输入：s = "([][{}])"
	输出: true
	```
	```
	输入：s = "([[{}]"
	输出: false
	```
```

java 演示代码
```java
import java.util.Stack;

class Solution {
	public boolean isValid(String str) {
		Stack<Character> stack = new Stack<Character>();
		char[] s = str.toCharArray();
		boolean flag = true;
		for (int i = 0; i < s.length; i++) {
			if (s[i] == ']' || s[i] == '}' || s[i] == ')') {
				if (stack.size() == 0) {
					flag = false;
					break;
				}
				char t = stack.pop();
				if (s[i] == ']' && t != '[' || s[i] == '}' && t != '{' || s[i] == ')' && t != '(') {
					flag = false;
					break;
				}
			} else {
				stack.push(s[i]);
			}
		}
		if (stack.size() != 0) {
			flag = false;
		}
		System.out.println(flag);
	}
}
```
